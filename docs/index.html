Warum Foxly MOTD?

Informationen, nicht Wartezeit.

Netzwerk- und APT-Aufgaben laufen über systemd im Hintergrund. Beim Login liest das MOTD nur lokale Daten.

⚡

Sofortige Ausgabe

Kein apt-get update, kein API-Aufruf und kein Warten im Login-Pfad.

🔄

Sichere Updates

Versionierte Releases, SHA-256-Prüfung, Backups und ein integrierter Rollback.

🧩

Sauber konfiguriert

Farben, Docker, Paketstatus und Update-Modus lassen sich zentral steuern.

Installation

In einer Minute startklar.

Das Installationsskript lädt das aktuelle Release, prüft seine SHA-256-Summe und aktiviert die Cache-Timer.

curl -fsSL https://motd.foxly.de/install.sh | sudo bash
foxly-motd statusInstallation und Timer prüfen
sudo foxly-motd updateAktuelles Release installieren
foxly-motd previewMOTD direkt anzeigen
sudo foxly-motd rollbackLetztes Backup wiederherstellen
